### :rocket: Project
**Be the Hero** is a project **mobile** and **web** developed with the intention of **helping** charities financially, **helping** with the costs of incidents.

### :mortar_board: Instructions
**Clone** the repository
```bash
git clone git@github.com:vargastm/be-the-hero.git
```

<br>

To run **backend** - Open the **backend** folder with the terminal and **run** the commands:
```bash
npm install
npm start
```
> Server will be running on localhost:3333

<br>

To run **frontend** - Open the **frontend** folder with the terminal and **run** the commands:
```bash
npm install
npm start
```
> Server will be running on localhost:3000

<br>

To run **mobile** - Open the **mobile** folder with the terminal and **run** the commands:
```bash
npm install
npm start
```
> If you have "Expo" installed on your phone, just scan the QRCode and the app will run in your phone.
